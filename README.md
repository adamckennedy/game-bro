# gameMuffin

## Purpose
Find your favorite games and review them on this personal Library!
* A user can signup/ login and add/ comment on their favorite games

## Built With
* node.js
* express.js
* Materialize CSS 
* JavaScript
* Sequelize
* mysql2
* handelbars.js
